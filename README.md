# Professional Worker Mental Health

A data analysis project exploring mental health among professional workers, based on the Depression Survey/Dataset for Analysis.

## Overview

This project analyzes survey data on workplace mental health, exploring potential relationships between depression risk and factors such as job stress, lifestyle habits, and demographics.

## Data Source

Dataset from Kaggle: Depression Survey/Dataset for Analysis

## Project Contents

- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature distribution visualizations
- Analysis of factors associated with depression risk

## Getting Started

1. Clone this repository
```bash
git clone https://github.com/chitaxiang/mental_health_project.git
```

2. Open the Jupyter Notebook to view the full analysis

## Requirements

- Python
- Jupyter Notebook
- pandas, numpy, matplotlib, seaborn (and other common data analysis libraries)
